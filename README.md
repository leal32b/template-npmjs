## Installation
> npm install fictum

```javascript
// javascript
const templateNpmjs = require('template-npmjs')
```

```typescript
// typescript
import templateNpmjs from 'template-npmjs'
```

```typescript
// Calling example1 function
templateNpmjs.greeter('Johnn Doe')  // Hello John Doe!
```
